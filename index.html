Memory game

Rules:

In the Memory Game, the objective is to find and match all pairs of identical cards. The game begins by shuffling the deck and laying all the cards face down in a grid. Players take turns flipping over two cards. If the cards match, the player keeps the pair and takes another turn. If the cards don't match, they are flipped back over, and the next player takes their turn. The game continues until all pairs have been matched. The player with the most pairs at the end of the game wins. If players have an equal number of pairs, the game ends in a tie.
